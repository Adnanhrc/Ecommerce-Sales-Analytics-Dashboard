# 🛒 📊 E-commerce Sales Analytics Dashboard (Power BI)

## 📌 Overview

This project is an interactive **E-commerce Sales Analytics Dashboard** built using **Power BI** to analyze sales performance, customer behavior, and profit trends.

It provides business insights into **sales, quantity, profit, and payment patterns**, helping organizations make data-driven decisions.

---

## 🚀 Key Features

* Sales, profit, and quantity performance tracking
* Customer and state-wise sales analysis
* Product category and sub-category insights
* Payment mode analysis
* Monthly and quarterly profit trends
* KPI cards for quick business overview

---

## 📊 Dashboard Insights

* 💰 Total Sales Amount: **161K**
* 📦 Total Quantity Sold: **2008**
* 📈 Total Profit: **26K**
* 💳 Average Order Value (AOV): **44K**

📌 Key observations:

* **Maharashtra, Madhya Pradesh, and Delhi** are top-performing states
* **Printers and Phones** generate the highest profit among sub-categories
* Majority of transactions are through **Cash on Delivery (~43%)**
* Sales distribution is heavily concentrated in a few customers
* Profit trends vary significantly across months

(Data sourced from dashboard visualization )
